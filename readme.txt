### Shortcodes included with Bus Ticket Booking with Seat Reservation

**To display the search form:**  
`
[wbtm-bus-search-form]
`
This shortcode allows you to add a search form to any page where customers can search for bus tickets and buy ticket
if someone like to add left filter option then need to add new parameter for left-filter
`
[wbtm-bus-search-form left_filter='on' left_filter_type='on' left_filter_operator='on' left_filter_boarding='on']
`
We have a dummy import option and guidelines. if you check the welcome link left sidebar of our plugin. you will dummy import guidelines.

Upload the Bus Ticket Booking with Seat Reservation to your blog, Activate it, then use shortcode as your need). Or Search Bus Ticket Booking with Seat Reservation from Plugin -> Add New then click on Install then click on Active.

= What kind of payment method works =
Our plugin supports all types of payment methods available for WooCommerce. 


= Is Bus Ticket Booking with Seat Reservation is Free? =
